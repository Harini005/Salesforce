git clone                               -----       Clones the existing git repository in the Selected folder
git add *                               -----       Stages all the changed files 
git add <fileName>                      -----       Stages Only the Specific fileName
git commit -m "message"                 -----       Adds the commit message
git push origin <branchName>            -----       Pushese the commited changes to the branch
git status                              -----       Compares the branch changes against workSpace
